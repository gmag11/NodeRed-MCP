## Why

The Node-RED Cookbook documents well-established flow patterns (HTTP endpoints, MQTT subscribers, data routing, error handling) that cover the majority of real-world use cases. Translating these patterns into MCP tool sequences gives the LLM ready-made recipes to apply, drastically reducing the number of reasoning steps required to build common flows.
